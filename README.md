# NeoNet Miner

**Earn NNET tokens by providing AI compute power to the NeoNet network.**

## Quick Start

```bash
git clone https://github.com/neonetainetwork/neonet-miner.git
cd neonet-miner
pip install aiohttp numpy
python neonet_miner.py --wallet neo1your_wallet_address
```
